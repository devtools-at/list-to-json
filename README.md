## What is List to JSON?

Convert line-separated lists to JSON arrays. This is a standalone TypeScript/JavaScript implementation that you can use in your own projects.

## Features

- **Zero Dependencies** - Lightweight with no external dependencies
- **TypeScript Support** - Full type definitions included
- **Browser & Node.js** - Works in both environments
- **UTF-8 Support** - Handles international characters correctly
